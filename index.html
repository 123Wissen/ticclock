Tic Clock

Digital Clock

How to use:
  • • Click "12/24 Hour" to switch time format
  • • Toggle analog clock display with "Show/Hide Analog"
  • • Enable speaking clock with the speaker icon

Alarms

How to use:
  • • Click "Add" to set a new alarm
  • • Choose time and ringtone
  • • Toggle alarms on/off with the bell icon
  • • Delete alarms with the trash icon

Timer

How to use:
  • • Enter hours, minutes, and seconds
  • • Click "Start" to begin countdown
  • • Pause/Resume anytime
  • • Reset timer with "Reset" button
  • • Notification when timer completes
00:00:00

Stopwatch

How to use:
  • • Click "Start" to begin timing
  • • Record lap times with "Lap" button
  • • Pause/Resume anytime
  • • Reset to zero with "Reset"
  • • View lap history below
00:00:00.000

World Clock

How to use:
  • • Click "Add" to add a new timezone
  • • Select city/timezone from the list
  • • Remove clocks with the × button
  • • Times update automatically

Pomodoro

How to use:
  • • Select timer type: Focus (25min), Short Break (5min), or Long Break (15min)
  • • Click "Start" to begin session
  • • Take short breaks after each focus session
  • • Take long break after 4 focus sessions
  • • Notification when each session ends
